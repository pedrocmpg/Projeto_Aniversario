# 🎂 Projeto Aniversário

Um site interativo e personalizado desenvolvido como presente de aniversário para minha namorada. Este projeto nasceu da vontade de aprender tecnologias de **Frontend** de forma prática, servindo como um descanso criativo do desenvolvimento do meu projeto principal, o GYMNight.

## 🚀 Objetivo
O foco principal é o aprendizado de **JavaScript (Vanilla)** para manipulação de DOM e lógica de tempo, além de consolidar conhecimentos de **HTML5** e **CSS3 (Flexbox/Grid)** sem o uso de frameworks.

## 🛠️ Tecnologias
* **HTML5:** Estrutura semântica da página.
* **CSS3:** Estilização moderna e layout responsivo.
* **JavaScript (ES6+):** Lógica da contagem regressiva e interatividade.
* **GitHub Pages:** Hospedagem automatizada do projeto.

## 📈 Funcionalidades Planejadas
- [ ] **Contador Regressivo:** Cronômetro em tempo real do dia que nos conhecemos.
- [ ] **Galeria de Fotos:** Um carrossel interativo com momentos especiais.
- [ ] **Mensagens Dinâmicas:** Sistema de sorteio de frases via JavaScript.
- [ ] **Design Responsivo:** Adaptado para visualização em dispositivos móveis.
